# Дашборд (docs/)

Статический SPA без сборки: `docs/index.html` + `docs/assets/app.js` + `docs/assets/style.css`. Корень GitHub Pages — `docs/`. Дизайн-система — **Neural Radar** (ADR-6): токены в начале `style.css` (эталон — `design_example/design-tokens.css`), шрифты Orbitron/Rajdhani/Inter/JetBrains Mono с Google Fonts, тёмная (по умолчанию) и светлая темы через `data-theme` на `<html>`.

## Структура

- **Шапка** (sticky): логотип с анимированным радаром, индикатор «обновлено» с датой-временем последнего обновления (generated_at свежайшего дневного файла, в локальном времени зрителя), селектор даты (виден всегда: в «Дне» выбирает день, в «Неделе»/«Месяце» — конец периода), **срезы День/Неделя/Месяц** (пилюли), переключатели RU/EN и темы. Состав шапки не меняется при переключении срезов.
- **Вкладки** (sticky под шапкой): «Радар дня» (новости) и «Эксперты · статьи и видео». Срез из шапки действует на обе вкладки.
- **Hero**: радар + счётчики (новостей, макс. impact, сюжетов, конспектов) за выбранный период.

## Как работает

- При старте грузит `data/index.json`, затем дневные `data/daily/<date>.json` по мере надобности (кэш в памяти).
- **Срезы** считаются на клиенте: `aggregate()` группирует по `cluster_id`, в кластере оставляет запись с максимальным `impact_score` (при равенстве — свежайшую), топ-N (неделя 10, месяц 15).
- **Карточка новости**: severity-бейдж по тиру (1–3 Минор / 4–6 Заметно / 7–8 Мажор / 9–10 Парадигма; цветная верхняя кромка карточки), вендорский чип в фирменном цвете (`VENDORS` в app.js ↔ `tag` в config/sources.yaml, цвета — токены `--vendor-*`), номер, сводка, курсивное обоснование оценки, первое мнение как quote-блок, остальные мнения и кейсы — в `<details>`, градиентный impact-бар с оценкой, футер с источником.
- **Карточка эксперта**: обложка (`cover` из данных — кадр видео; без обложки — глиф типа), бейдж Статья/Видео, автор+дата, сводка, ссылки «Конспект» (note_page) и «Оригинал».
- **Фильтры по моделям** — чипы на вкладке «Радар» (localStorage, влияют только на отображение).
- **RU/EN и темы** — localStorage (ключи `lang`, `theme`), общие со страницами конспектов. Все строки UI — словарь `I18N` в app.js; контент — двуязычные поля данных.
- **URL-параметры**: `?view=day|week|month&date=YYYY-MM-DD&lang=ru|en&theme=dark|light&tab=radar|experts`.
- Баннер «демо-данные», если у загруженного дня `demo: true`.

## Страницы конспектов (docs/notes/)

Отдельные HTML с общими `note.css`/`note.js`; двуязычность через `<span data-lang="ru|en">` + `data-lang-active` на `<html>`. Формат (v1.1): **связный текст** 3–6 абзацев с подзаголовками; методы описываются развёрнуто; таймкоды видео — инлайн-ссылки `[MM:SS]`; кадры — `<figure>` рядом с относящимся абзацем. Образцы: страницы за 2026-07-09. Тема Neural Radar применяется к конспектам через **алиасы старых переменных** (`--surface`, `--ink`, `--accent`… → токены) в конце `style.css` — HTML конспектов при смене дизайна не менялся; классы `.topbar`/`.chip-btn` стилизованы там же.
